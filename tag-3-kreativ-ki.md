## 🖼️ Kapitel 1: KI-Bildgenerierung verstehen

### Wie entsteht ein KI-Bild?

Stellt euch vor, ihr beschreibt einem Maler ein Bild:
> "Male einen Creeper in einem Sonnenuntergang am Strand"

Der Maler malt das nach seinen Vorstellungen. Genau so arbeitet eine KI-Bildgenerator!

### Beispiele mit Minecraft:

**Prompt 1:**
```
A Minecraft creeper standing on a beach at sunset, 
pixel art style, beautiful lighting
```
→ Die KI kombiniert: Creeper + Strand + Sonnenuntergang + Pixel-Look

**Prompt 2 (viel detaillierter):**
```
A detailed Minecraft scene: Steve and Alex building 
a castle together, blue sky with fluffy clouds, 
blocky art style, vibrant colors, cinematic angle, 
4K quality, concept art
```
→ Viel spezifischer, besser kontrolliertes Ergebnis!

## 🎮 Roblox & Minecraft: Skin-Design mit KI

### Aufgabe: Eigener Minecraft-Skin entwerfen

**Schritt 1: Konzept mit KI entwickeln**
```
ROLLE: Du bist ein Minecraft-Skin-Designer.

AUFGABE: Entwirf einen einzigartigen Skin für einen 
         12-jährigen Abenteurer namens "Maxim".

KONTEXT: 
- Er liebt Weltraum und Raketen
- Seine Lieblingsfarbe ist Blau
- Er soll cool aber nicht gruselig aussehen
- Der Skin soll im Multiplayer auffallen

FORMAT: Beschreibe den Skin detailliert:
1. Kopf (Gesicht, Kopfbedeckung?)
2. Oberkörper (Shirt, Details)
3. Unterkörper (Hose, Gürtel?)
4. Schuhe
5. Besondere Features (Cape, Overlay?)

Gib mir HEX-Farbcodes für die Hauptfarben!
```

**Schritt 2: Mit Tools umsetzen**
- geht zu: minecraftskins.com oder novaskin.me
- Nutzt die KI-Beschreibung als Vorlage
- Baut den Skin Schritt für Schritt

## 📚 Kapitel 2: Geschichten schreiben mit KI

### Das "Choose Your Own Adventure" Prinzip

Kennt ihr Bücher, bei denen ihr entscheidet, wie die Geschichte weitergeht? Genau das könnt ihr mit KI erstellen!

### Beispiel für Maxim, Alia und Amir:

```
ROLLE: Du bist ein Kinderbuchautor für Abenteuergeschichten.

AUFGABE: Schreibe den Anfang einer Geschichte für drei 
         Geschwister namens Maxim, Alia und Amir.

KONTEXT:
- Die drei finden ein geheimnisvolles Portal in ihrem Keller
- Das Portal führt in die Welt von Minecraft
- Jeder hat eine besondere Fähigkeit:
  * Maxim kann mit Redstone umgehen (Technik)
  * Alia ist eine tolle Baumeisterin (Kreativität)
  * Amir ist mutig und gut im Kampf (Abenteuer)

FORMAT: 
- Schreibe 3 Absätze Einleitung
- Dann präsentiere eine Entscheidung:
  "Was tun die drei?"
  A) Sie betreten das Portal sofort
  B) Sie rüsten sich erst aus
  C) Sie rufen einen Erwachsenen

Warte auf meine Antwort, bevor du weiter schreibst!

STIL: Spannend, kindgerecht, humorvoll
```

### Die Interaktive Geschichte:

**Maxim wählt A) Portal betreten**
→ Die KI schreibt die Fortsetzung mit dieser Entscheidung!

**Alia möchte B probieren**
→ Gleicher Anfang, andere Fortsetzung!

## 🎪 Übung: Eigene Roblox-Game-Story

### Aufgabe:
Entwickelt eine Hintergrundgeschichte für ein Roblox-Spiel!

```
ROLLE: Du bist Narrative Designer für Roblox-Spiele.

AUFGABE: Erstelle eine fesselnde Hintergrundgeschichte 
         für ein neues Roblox-Spiel.

KONTEXT:
- Genre: Mystery/Abenteuer
- Setting: Eine verlassene Roblox-Spielwelt, die 
  langsam verblasst (Glitch-Effekte)
- 3 Hauptcharaktere basieren auf den Spielern 
  (Maxim, Alia, Amir)
- Das Spiel heißt "The Forgotten Lobby"

FORMAT:
1. PREMISE (2 Sätze: Was ist das Spiel?)
2. DIE WELT (Beschreibung der verblassten Welt)
3. CHARAKTERE (3 Kurzbeschreibungen mit Spezialfähigkeiten)
4. DAS ZIEL (Was müssen die Spieler erreichen?)
5. TWIST (Eine überraschende Wendung)
6. SPIELMECHANIK-IDEEN (Wie wird die Story im Spiel erzählt?)

TON: Geheimnisvoll, spannend, aber nicht gruselig (Alter 12+)
```

## 🧪 Experiment: Der Comic-Ersteller

### Ziel: Einen Minecraft-Comic erstellen!

**Schritt 1: Storyboard mit KI planen**
```
ROLLE: Du bist Comic-Autor für Minecraft-Comics.

AUFGABE: Plane einen 6-seitigen Comic über ein 
         Minecraft-Abenteuer.

KONTEXT:
- Hauptcharakter: Ein Creeper namens "Carl"
- Carl will keine Monster mehr sein, sondern Freunde finden
- Setting: Ein Dorf voller Villager

FORMAT: Für jede Seite gib an:
- Szene (Was sieht man?)
- Dialog/Text
- Emotion/Stimmung

STIL: Lustig, herzerwärmend, Minecraft-typisch
```

**Schritt 2: Illustrationen**
- Entweder: KI-Bildgenerator nutzen (z.B. Bing Image Creator, Midjourney)
- Oder: Selbst zeichnen mit der KI-Beschreibung als Vorlage!

**Schritt 3: Zusammenstellen**
- Canva, PowerPoint oder einfach auf Papier
- Text einfügen, Layout gestalten

```

- Chorus

STIL: Lustig, rhythmisch, Minecraft-Referenzen (Creeper, 
      Diamanten, Enderman), kindgerecht
```

## 🎯 Praxisprojekt: Das Familien-Comic-Universum

### Ziel:
Erstellt gemeinsam ein Comic-Universum mit euren Avataren!

### Phase 1: Charakter-Design (30 Min)
Jeder entwirft seinen eigenen Charakter:

**Maxim's Charakter:**
```
Name: ________________
Aussehen: ________________
Spezialfähigkeit: ________________
Persönlichkeit: ________________
```

**Nutzt die KI für Ideen:**
```
Hilf mir, einen Charakter zu entwerfen.
Name: [eurer Name]
Alter: 12
Interessen: [eure echten Interessen]
Setting: Ein Roblox-Spiel über __________

Beschreibe:
- Aussehen
- Outfit
- Persönlichkeit
- Spezialfähigkeit
- Schwäche (jeder hat eine!)
```

### Phase 2: Die Welt (20 Min)
```
Wir haben drei Charaktere: [Beschreibungen einfügen]

Erstelle eine Welt, in der alle drei zusammenpassen:
- Was für eine Welt ist es?
- Was ist das Ziel?
- Wer ist der "Bösewicht" oder das Problem?
- Wie arbeiten die drei zusammen?
```

### Phase 3: Erste Geschichte (30 Min)
Nutzt die interaktive Geschichte aus Kapitel 2 und spielt sie!

## 💬 Eltern-Notiz

**Wichtiger Aspekt:**
KI ist ein Kreativ-Werkzeug, keine Ersatzkreativität. Die Kinder sollten verstehen:
- Die Idee kommt von ihnen
- Die KI ist nur der "Pinsel"
- KI-generierte Inhalte haben rechtliche Grenzen (keine kommerzielle Nutzung ohne Prüfung!)
